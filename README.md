# Raycaster-3JS

An interactive 3D raycaster built using Three.js, featuring a dynamic scene with rotating icosahedrons and animated particles. Click on objects to focus the camera on them and explore the 3D space.

## Demo

Experience the raycaster in action: [Live Demo](https://binaryvortex.github.io/Raycaster-3JS/)

## Features

- **Raycasting Interaction**: Click on icosahedrons to highlight and move the camera towards them.
- **Animated Elements**: Rotating icosahedrons and floating particles for a dynamic visual experience.
- **Mouse Controls**: Move the mouse to rotate the entire scene.
- **Responsive Design**: Adapts to different screen sizes.
- **Fog and Lighting**: Atmospheric fog and multiple light sources for depth.

## Technologies Used

- [Three.js](https://threejs.org/) - 3D graphics library
- [GSAP](https://greensock.com/gsap/) - Animation library
- [Bootstrap](https://getbootstrap.com/) - CSS framework

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/BinaryVortex/Raycaster-3JS.git
   ```
2. Open `index.html` in your web browser.

Alternatively, visit the [live demo](https://binaryvortex.github.io/Raycaster-3JS/) to see it in action.
